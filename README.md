# Beeper v1.0
##Made with ArduinoIDE

Limbus Company–inspired beeper using M5GFX.

## Pages
- Welcome
- Prescript
- Hermes (Yes/No)
- Tyche (D20 roll)

## Controls
- BtnA → action  
- BtnB → switch page  

## Hardware
- M5StickS3  
- Any ESP32 device compatible with M5GFX (may require adjustments)
