Beeper by Yumi v1.0

Limbus Company inspired beeper for M5StickS3 device using M5GFX.

Pages:
- "Home/Welcome page"
- "Prescript"
- "_Hermes._" (yes/no)
- "_Tyche._" (D20 roll)

Controls:
BtnA → action
BtnB → switch page

Hardware:
- M5StickS3
- Any esp32 device that's M5GFX compatible(might require tweaking)
